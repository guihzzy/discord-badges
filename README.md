# Discord Badges 🏆

Uma coleção apresentando todas as insígnias do Discord!

As insígnias do Discord são conquistas de aparência épica para deixar seu perfil mais legal. Elas são concedidas pelo Discord por vários motivos, como ser um impulsionador Nitro, ser um parceiro do Discord ou ser um membro da equipe do Discord. Esta coleção apresenta todas as insígnias do Discord tanto para contas de equipe/bots quanto para contas de usuários. Foi criada por **dominikdev**

## Hall da Fama

Uma coleção de eventos únicos que aconteceram no Discord.

| Explicação | Imagem |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------- |
| O APP **Spongetext** (usado para demonstrar o novo recurso de [Apps Instaláveis por Usuário](https://discord.com/developers/docs/tutorials/developing-a-user-installable-app) do Discord) recebeu a insígnia **Equipe do Discord**, embora não esteja claro por que o bot recebeu isso | <img  src="/assets/fame/spongetextstaff.png"> |
| Devido ao fechamento do servidor de Ex-Alunos de Moderadores, Sapphire e GearBot foram um dos únicos APPS a obter a insígnia **Ex-Aluno dos Programas de Moderador** | <img  src="/assets/fame/sapphiremod.png"> <img  src="/assets/fame/gearmod.png"> |
| Há algum tempo, muito antes do Clyde, o Discord criou um chatbot semelhante ao que o Clyde se tornou | <img  src="/assets/fame/wumpus.png"> |
| Em fevereiro de 2019, o Discord se uniu ao King of the Hat para criar uma insígnia temática do HypeSquad Equilíbrio. Qualquer pessoa na casa HypeSquad Equilíbrio pôde manter a insígnia por um mês. King of the hat foi o primeiro jogo a ser trazido para o Discord e foi gratuito por uma semana. Uma [competição](https://medium.com/king-of-the-hat/hat-is-free-this-week-and-this-week-only-v-f9fa0987688b) entre as casas do HypeSquad também foi realizada, com um PC personalizado sendo sorteado. O usuário DPS Abook, que representava o HypeSquad Brilhante, venceu a competição | <img src="/assets/kothhypesquadbalance.svg"> |
| Para o [Snowsgiving](https://discord.fandom.com/wiki/Snowsgiving) 2019, os usuários na casa HypeSquad Equilíbrio tiveram sua insígnia alterada de verde para dourado. O Discord realizou uma competição de arrecadação de dinheiro que resultou na Casa do Equilíbrio vencendo e a insígnia foi temporariamente dourada até 3 de fevereiro de 2020. Esta é a única insígnia animada que existiu no Discord. Ela brilhava a cada poucos segundos | <img  src="/assets/goldenhypesquadbalance.svg"> |
